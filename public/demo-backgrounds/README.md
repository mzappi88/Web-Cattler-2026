# Demo Background Images

Esta carpeta contiene las imágenes de fondo para la página `/demo`.

## Estructura de archivos:

- `bunkscore.jpg` - Imagen de fondo para **todos los casos** (mobile y desktop, todos los idiomas)

## Lógica de selección:

### Todos los casos:

- **Cualquier idioma/país**: `bunkscore.jpg`
- **Mobile y Desktop**: `bunkscore.jpg`

## Layout Desktop:

- **Banner superior**: Imagen de 320px de altura (`h-80`) para mostrar las dos personas
- **Formulario**: Posicionado a la derecha
- **Texto catchy**: A la izquierda con "Start Your Free Demo" y "No Credit Card Necessary"

## Cómo agregar imágenes:

1. Coloca las imágenes en esta carpeta con los nombres exactos
2. Asegúrate de que las imágenes tengan buena resolución y sean apropiadas para fondos de página web
3. El código usa automáticamente `bunkscore.jpg` para todos los casos

## Formatos recomendados:

- JPG para fotografías
- WebP para mejor compresión
- PNG para imágenes con transparencia (aunque no es común para fondos)
